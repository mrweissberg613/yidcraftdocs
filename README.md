# Yid Craft Documentation

Welcome to the Yid Craft docs site. This workspace is built with Astro + Starlight and contains the documentation, guides, and roadmap for Yid Craft.

## 📚 What’s inside

- `src/content/docs/` — documentation content and site pages
- `src/content/docs/guides/` — step-by-step guides for authors and maintainers
- `src/content/docs/reference/` — reference pages, roadmap, and future planning
- `src/assets/` — images and visual assets used by the docs

## 🚀 Common commands

Run these from the project root:

| Command                   | Action                                           |
| :------------------------ | :----------------------------------------------- |
| `npm install`             | Install dependencies                             |
| `npm run dev`             | Start local dev server                            |
| `npm run build`           | Build the production site                         |
| `npm run preview`         | Preview the built site locally                    |
| `npm run astro -- --help` | Help with Astro CLI commands                      |

## ✨ Want to update the wiki?

Edit pages under `src/content/docs/`, then refresh the dev server. New files are automatically turned into routes.

If you want to add future sections, create a page under `guides/` or `reference/` and link it from the homepage.
