The Homes system allows you to save locations and teleport back to them whenever you need.

> You may have a maximum of **3 homes** at any time.

## Using the Home GUI

<img src="/assets/homegui.png" alt="Home GUI" width="400" height="400" />

```bash
/home
```

This command opens the Home GUI, where you can create, manage, and teleport to your homes.

### Creating a Home

To create a new home:

- Click the **Bed** icon in the Home GUI, or
- Run the following command:

```bash
/home create <home_name>
```

### Teleporting to a Home

To teleport to one of your saved homes:

1. Open the Home GUI.
2. Click the **Oak Door** icon to view your homes.

<img src="/assets/homemenu.png" alt="Home Menu" width="400" height="400" />

3. Left-click on the home you want to teleport to.

### Deleting a Home

To delete a home:

- Right-click the home in the Home Menu, or
- Run the following command:

```bash
/home delete <home_name>
```
