---
title: Teams System
description: Learn about the Yidcraft Teams system and how it works.
---

Teams let you group up with other players to build, play, and progress together. You can share money, manage territory, and work as a team.

Your current team is displayed on the scoreboard

---

## Basic Commands

```bash
/team create <name>
```
Create a new team.

```bash
/team join <team>
```
Join a team.

```bash
/team leave
```
Leave your current team.

```bash
/team info [team/player]
```
View information about a team or a player’s team.

```bash
/team help
```
Shows the commands you can use.

---

## Team Chat

```bash
/team chat [message]
```
Send a message to team chat. If no message is included, it toggles team chat mode.

```bash
/team allychat [message]
```
Send a message to allied teams.

---

## Homes and Warps

```bash
/team home
```
Teleport to your team home.

```bash
/team sethome
```
Set your team home.

```bash
/team delhome
```
Remove your team home.

```bash
/team warp <name> [password]
```
Teleport to a team warp.

```bash
/team setwarp <name> [password]
```
Create a team warp.

```bash
/team delwarp <name>
```
Delete a team warp.

---

## Team Economy

```bash
/team bal
```
Check your team balance.

```bash
/team deposit <amount>
```
Deposit money into the team balance.

```bash
/team withdraw <amount>
```
Withdraw money from the team balance.

---

## Rankings

```bash
/team top
```
View top teams.

```bash
/team baltop
```
View richest teams.

```bash
/team rank [team]
```
Check a team’s rank.

---

## Other Features

```bash
/team echest
```
Open the team ender chest.

```bash
/team pvp
```
Toggle PvP between teammates.

---

## Team Management (Leader Only)

These commands can only be used by the team owner.

```bash
/team invite <player>
/team kick <player>
/team ban <player>
/team unban <player>
```

```bash
/team promote <player>
/team demote <player>
/team setowner <player>
/team title <player/me> <title>
```

```bash
/team name <name>
/team description <description>
/team color <color>
/team tag
/team open
```

---

## Disbanding a Team (Leader Only)

Only the team owner can do this:

```bash
/team disband
```

This permanently deletes the team.
